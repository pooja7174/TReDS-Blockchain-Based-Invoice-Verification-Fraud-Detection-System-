# TReDS-Blockchain-Based-Invoice-Verification-Fraud-Detection-System

A blockchain-powered FinTech project designed to enhance transparency, security, and trust in the TReDS (Trade Receivables Discounting System) ecosystem by enabling secure invoice verification and fraud detection through immutable financial records.

# Project Overview

This project was developed as part of our FinTech Module, focusing on the implementation of Blockchain Technology in TReDS platforms to address major financial challenges such as invoice duplication, fraudulent transactions, data tampering, and lack of transparency in receivables financing systems.

The system securely stores invoice records on a blockchain network, where each invoice is converted into a block containing unique cryptographic hashes. Since blockchain data is immutable, any unauthorized modification instantly changes the hash value, making fraudulent activities easily identifiable.

Our objective was to build a secure and transparent solution that improves trust between buyers, sellers, and financial institutions within the TReDS ecosystem.

# Key Features

🔐 Blockchain-Based Invoice Verification

Each invoice is stored as a secure block containing:

Invoice details

Timestamp

Current hash value

Previous block hash


This creates a tamper-proof chain of records that ensures invoice authenticity and data integrity.

⚠️ Fraud Detection Mechanism

The system validates invoices through hash verification techniques. Any change in invoice data results in a different hash, allowing immediate detection of manipulated or fraudulent invoices.

📊 TReDS Invoice Status Tracking

The platform enables real-time tracking of invoice status such as:

Pending

Verified

Rejected


This improves operational transparency and streamlines financial workflows.

📧 Automated Email Notifications

Integrated Gmail API for sending automated notifications regarding invoice verification and status updates to users.

🛡️ Immutable Financial Records

Blockchain ensures that once invoice data is added to the chain, it cannot be altered without affecting the entire network structure, providing reliable and secure financial record management.

💻 User-Friendly Dashboard

An interactive interface was designed for smooth invoice submission, verification, and monitoring processes.
